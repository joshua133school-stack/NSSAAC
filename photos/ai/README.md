# AI-generated photos

Put your **AI-generated** images in this folder.

- Accepted types: `.jpg .jpeg .png .gif .webp .bmp .avif`
- The filename doesn't matter — the server discovers files automatically.
- For a fair study, try to roughly match the number of images in `../real/`
  and keep the subject matter and quality similar across both folders
  (e.g. don't make all AI ones portraits and all real ones landscapes).
- Aim for at least 10–15 here so each participant sees a good variety.

> Note: image URLs include this folder name, so a participant who inspects
> their browser's network tab could in principle see which folder an image
> came from. For an honest, anonymous study this is fine.
