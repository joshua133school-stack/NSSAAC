# Real (non-AI) photos

Put your **real photographs** (taken with a camera, not AI-generated) in this
folder.

- Accepted types: `.jpg .jpeg .png .gif .webp .bmp .avif`
- The filename doesn't matter — the server discovers files automatically.
- Try to roughly match the count in `../ai/`, with similar subject matter and
  image quality so the comparison is fair.
- Aim for at least 10–15 here.
